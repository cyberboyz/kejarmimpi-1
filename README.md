# km-gingonic

# Deps
* install golang
* set gopath

# Running in local
* clone this repo
* run by: `go run server.go`
* open localhost:8080

# Running in Heroku
* clone this repo
* heroku login
* heroku create // create apps in heroku
* heroku addons:create heroku-postgresql:hobby-dev // add database in heroku
* git push heroku master
* heroku open
